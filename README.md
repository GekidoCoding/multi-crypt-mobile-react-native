# Application Mobile Multi-Crypto

## Description générale

L’**application mobile Multi-Crypto** est une composante du projet global **Multi-Crypto**. Elle est destinée aux utilisateurs finaux et permet le **suivi en temps réel des mouvements et des variations des crypto-monnaies**.

Cette application offre une interface mobile intuitive et accessible, facilitant la consultation des informations liées aux changements du marché des crypto-monnaies.

---

## Fonctionnalités principales

L’application mobile permet notamment :

* Le suivi des **variations et mouvements des crypto-monnaies**
* La consultation des évolutions de prix
* L’accès aux informations essentielles du marché crypto
* Une visualisation claire et simplifiée des données
* Une interaction sécurisée avec le système Multi-Crypto via des API

---

## Interaction avec le système Multi-Crypto

L’application mobile communique avec les **API sécurisées du projet Multi-Crypto**, notamment :

* Les API d’authentification et d’autorisation
* Les API de récupération des données liées aux crypto-monnaies

Cette communication garantit la fiabilité, la sécurité et la cohérence des informations affichées à l’utilisateur.

---

## Sécurité

La sécurité des échanges est assurée par :

* Une authentification sécurisée des utilisateurs
* Des communications protégées avec les API
* Le respect des bonnes pratiques en matière de sécurité mobile

---

## Technologies utilisées

* **Type d’application** : Application mobile
* **Architecture** : Client – API REST
* **Backend** : API RESTful Multi-Crypto (Symfony et Spring boot)
* **Plateforme mobile** : (Android / iOS à préciser si nécessaire)

---

## Objectif du projet

L’objectif de l’application mobile Multi-Crypto est de fournir aux utilisateurs un **outil simple, rapide et fiable** pour suivre les mouvements et changements du marché des crypto-monnaies, directement depuis leur appareil mobile.
